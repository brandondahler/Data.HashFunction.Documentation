Blake2B
Initializes an instance of the Blake2B class with the default settings (no key, no salt, no personalization, and 512 output length).
Blake2B(Int32)
Initializes an instance of the Blake2B class with the provided hashSize (no key, no salt, no personalization). The hashSize must be 8 bits at the least and 512 bits at most, and the size in bits must be a multiple of 8.
Blake2B(Byte, Byte, Byte)
Initializes an instance of the Blake2B class with the default hash size. If not null, the key, salt, and personalization arguments will be applied to the hashing algorithm. The key parameter must be a byte sequence of at most 64 bytes. The salt parameter must be a byte sequence of exactly 16 bytes. The personalization parameter must be a byte sequence of exactly 16 bytes.
Blake2B(Int32, IEnumerableByte, IEnumerableByte, IEnumerableByte)
Initializes an instance of the Blake2B class with the provided hashSize. If not null, the key, salt, and personalization arguments will be applied to the hashing algorithm. The hashSize must be 8 bits at the least and 512 bits at most, and the size in bits must be a multiple of 8. The key parameter must be a byte sequence of at most 64 bytes. The salt parameter must be a byte sequence of exactly 16 bytes. The personalization parameter must be a byte sequence of exactly 16 bytes.
