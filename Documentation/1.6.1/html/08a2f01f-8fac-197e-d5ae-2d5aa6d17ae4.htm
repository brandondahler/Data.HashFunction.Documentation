Initializes an instance of the Blake2B class with the default hash size. If not null, the key, salt, and personalization arguments will be applied to the hashing algorithm. The key parameter must be a byte sequence of at most 64 bytes. The salt parameter must be a byte sequence of exactly 16 bytes. The personalization parameter must be a byte sequence of exactly 16 bytes.
